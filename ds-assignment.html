Assignment: Web Scraping & Data Analysis

In this assignment you will work with data from Haryana Police FIR Database. This is a an attempt by Haryana Police to make the FIRs publicly available and searchable.

In this assignment, I want you to scrape the latest 50 FIR reports from Gurugram Sector 56 Police Station from the website. The data has to include

  • FIR Number
  • FIR Date
  • Act
  • Act Section Number
  • Suspect Name (Section 7 of the FIR)
  • Suspect Relative Name (Section 7 of the FIR)

In addition, I want you to add a current date variable to the data frame.

Afterwards, I want you to do a data analysis on the data. What do you think is interesting about this data? Write a sentence about some interesting thing you have discovered by looking at the data.

For example, you might consider whether there is a difference in (suspected) crimes levels at different times doing the day, or whether (if your data permits) crimes peaks at particular days doing the week or in particular location.

The assignment has to be handed in as a zip of the codebase. The code should run without loading any external files. If you need external files please upload them to github and load them directly into your script.

Requirement:

  • Python 3
  • Beautiful Soup
